# Product Overview

## Product Name
OpenMineDash

## Purpose
A transparent, open-source mining dashboard and manager for macOS (Apple Silicon). Orchestrates legitimate mining software with full user control and transparency.

## Key Features
- Single-panel dashboard for mining management
- Multi-coin support via JSON plugin system (XMR, BTC, LTC)
- Real-time stats: hashrate, shares, CPU/temp monitoring
- Live log viewer with filtering
- Pool health checking
- Profile management (Home, Office, Low Power)
- Checksum verification for miner binaries

## Target Users
- Cryptocurrency enthusiasts with macOS devices
- Users who want transparent, controllable mining
- Developers interested in mining orchestration

## Security Principles
- No cryptojacking: explicit user consent required
- Default OFF: mining never starts automatically
- One-click stop always available
- No persistence or background processes
- Binary checksum verification
